# Metal Price Tracker

## 🚀 Live App
https://metal-price-tracker.streamlit.app/

## 📊 Features
- Daily automated scraping of metal prices
- Historical data tracking
- Interactive dashboard (Streamlit)
- Cloud automation using GitHub Actions

## 🛠 Tech Stack
- Python
- Selenium
- SQLite
- Streamlit
- GitHub Actions

## ⚙️ How it works
1. GitHub Actions runs scraper daily
2. Data stored in SQLite database
3. Streamlit dashboard displays updated data
